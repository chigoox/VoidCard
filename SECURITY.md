# Security Policy

## Reporting a Vulnerability

Email **security@ed5enterprise.com**. We respond within 2 business days.

Please include:
- A description of the issue and its impact.
- Steps to reproduce (PoC welcome).
- Your name / handle for credit, if desired.

## Safe Harbor

We will not pursue legal action against good-faith researchers who:
- Avoid privacy violations, data destruction, and service disruption.
- Do not exploit beyond what is necessary to demonstrate the issue.
- Give us reasonable time to remediate before public disclosure (90 days default).

## Supported Versions

Only the latest production deployment of `vcard.ed5enterprise.com` is supported.
